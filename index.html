Atividades

Nesta seção estão os links para envio das atividades avaliativas requisitadas em sala.

  1. Minha Primeira Página - Esta foi a primeira atividade onde o objetivo era modificar os textos e imagens deste link
  2. Meu Perfil - Esta foi a segunda atividade, onde criamos uma página para ser seu perfil, contendo uma imagem de avatar, letras de música, lista de jogos e séries favoritos entre outros hobbies.
  3. Montando um hambúrguer - Esta foi a terceira atividade, onde é necessário realizar um fork do repositório fe1-hamburguer e utilizar as imagens do repositório para montar o hambúrguer da forma que vocês mais gostam utilizando os métodos de posicionamento vistos em sala.
